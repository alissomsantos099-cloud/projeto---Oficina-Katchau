# projeto---Oficina-Katchau
Oficina Katchau – Sistema Web para Gestão Automotiva Aplicação completa para gerenciamento de clientes, veículos, serviços e ordens de serviço, com dashboard inteligente, CRUD completo, documentação acadêmica e integração com MySQL. Desenvolvido em PHP e Bootstrap, ideal para uso em oficinas mecânicas reais ou como projeto de demonstração técnica.
